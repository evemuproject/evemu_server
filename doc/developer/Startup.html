Server startup

  The server startup, a simple rundown.

  There are three singletons created as global variables,
sEntityList, a list of all entitys in the game.
sItemFactory, a factory for createing and managing all the items in the game.
sManager, a service manager that allows the client and server to interact. 

// Create global singletons.
EntityList &_sEntityList = EntityList::get();
ItemFactory &_sItemFactory = ItemFactory::get();
PyServiceMgr &_sManager = *(new PyServiceMgr( 888444 ));
